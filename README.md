Random Password Generator

Простой генератор случайных паролей на Python. Утилита создаёт указанное количество паролей заданной длины, использует только буквы и цифры (без специальных символов) и сохраняет результат в текстовый файл.
Особенности

    Генерация паролей из латинских букв (верхний и нижний регистр) и цифр
    Можно задать количество паролей и длину каждого пароля
    Результат сохраняется в .txt файл с одной строкой — один пароль
    Использует криптографически стойкий источник случайности (secrets)

Требования

    Python 3.6+ (рекомендуется Python 3.8+)

Установка

    Скачайте или клонируйте репозиторий:

bash
git clone <repo-url>
cd <repo-directory>

    (Опционально) Создайте и активируйте виртуальное окружение:

bash
python3 -m venv .venv
source .venv/bin/activate   # Linux / macOS
.venv\Scripts\Activate.ps1  # Windows PowerShell

Использование

Скрипт: password_gen.py

Примеры запуска:

bash
# По умолчанию: 10 паролей по 16 символов, файл с меткой времени
python3 password_gen.py

# Сгенерировать 5 паролей по 12 символов и сохранить в my_passwords.txt
python3 password_gen.py -n 5 -l 12 -o my_passwords.txt

# Короткая помощь
python3 password_gen.py -h

Параметры:

    -n, --number — количество паролей (по умолчанию 10)
    -l, --length — длина пароля (по умолчанию 16)
    -o, --out — имя выходного файла (по умолчанию passwords_YYYYMMDD_HHMMSS.txt)

Пример вывода

Файл my_passwords.txt:

Code
aK3j9L0sPq1Zx7W2
Z9m2Tq6bN4fYw8Er
...

Безопасность и рекомендации

    Скрипт использует модуль secrets, который подходит для генерации ключей и паролей.
    Храните сгенерированные пароли в защищённом месте (менеджер паролей).
    Не передавайте файлы с паролями по небезопасным каналам.
